# Admin & the three tech points

v71: admin is fully repurposed into a **tech point producer**; the old resource drain from admins was redistributed to other systems. Admin points can optionally consume paper.

Three knowledge point types: **administration, innovation, knowledge**. Administration points are used mainly for empire techs (needed for e.g. town hall levels — see [[population-thresholds]]).
