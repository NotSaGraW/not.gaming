# Buildings (rooms)

All 112 buildable rooms, listed alphabetically. The game's own build-menu grouping is inconsistent
across versions, so no grouping is imposed here — sort or search the table instead.

**Output** is what the room yields, from its data: a good (`INDUSTRY`/`GROWABLE`/`MINABLE`/`YEILD`),
a satisfied need (`SERVICE.NEED`, e.g. Bathing, Drink), a point type (`BOOST CIVIC_*` →
Administration / Innovation / Knowledge), or population (`RACE`). A dash means the file encodes no
output — the room stores, houses, defends, moves goods, or does upkeep. Output goods link to
[[goods]]. Descriptions are the game's own `DESC` (`text/room/`), unedited.

Not in this list, deferred to a dedicated card: production **recipes** (inputs→outputs) and
**build/upkeep costs**. Those are structured — mandatory floor cost plus optional placeable items,
ongoing upkeep, and whole-building upgrade tiers drawing on separate resources — and need room to
be shown correctly rather than a lossy column here.
